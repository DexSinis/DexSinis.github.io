    收费操作 #

    一、 订单收费 #

    1 点击课时包管理 选中对应的课时包（陪练课/精品课） 点击生成订单

    2 填入实际的支付金额和选中对应的学生 点击保存生成订单

    3 发送对应的二维码给对应的客户进行扫码支付

    4 点击订单记录 确认支付状态是否支付成功

    5 开具对应的收据给对应用户 对应的日期 姓名 课时包 金额 经手人
