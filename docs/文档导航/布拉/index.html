    1.收费操作
    收费操作 # 一、 订单收费 # 1 点击课时包管理 选中对应的课时包（陪练课/精品课） 点击生成订单 2 填入实际的支付金额和选中对应的学生 点击保存生成订单 3 发送对应的二维码给对应的客户进行扫码支付 4 点击订单记录 确认支付状态是否支付成功 5 开具对应的收据给对应用户 对应的日期 姓名 课时包 金额 经手人
    2.排课管理
    排课管理 # 一、 常规排课 # 1 在课程列表中点击新增课程 # 2 填写对应的课程信息（ps：注意区分精品课和陪练课，正常以后都选择正式课，应该由班主任安排） # 二、固定排课 （后面尽量使用固定循环计划） # 1 点开固定排课并且填写的课程信息，并且选择需要循环的时间段 # 如上图 胡芷晴 需要排 刘奕含 每周一12:30的课（从2月1日～3月31日） # 2 点击提交约课后可以看到相应的课程信息，生成对应的课表详情，需要做二次的校对课程是否正确 # 3 如果发现冲突或者不需要可以选择删除或者更换老师 # 4 确认无误后，点击确定约课 就会跳转课程列表 # 三、固定循环计划 # 1 在课程列表中点击新增循环计划 # 2 录入需要每月循环的学生的信息（ps：注意区分精品课和陪练课） # 如上图 测试学生-演示 需要排 每周二13:00的课 # 3 点击查询对应的学生，并且点击编辑老师 # 4 录入对应的老师信息 # 5 再次核对循环计划信息(PS：每月的15号会自动生成下一个月的课程) # 如上面计划等到16号 课程列表就回出现3月1日～3月31日 每周二 13:00的精品课课程 # 四、先选老师 # 1 填写对应的课程信息（PS：注意区分陪练课和精品课） # 2 选中老师空余的时间 # 3 点击提交约课 # 4 查看核对课程是否正确 # 5 点击提交约课跳转课程列表 #
    3.内部体验课时间管理
    内部体验课时间管理 # 一、 开放时间管理 # 1 点击新增，按照上班时间录入正常上班时间时间 # 2 按照周循环的方式录入内部老师的工作时间 如赵阳 周五 下午13点到晚上21点班次 # 二、内部老师和内部时间登记 # 1 点击编辑老师信息 # 2 标记内部老师 # 3 标记体验课老师 # 4 app是的排课时间都是不可约的状态（如果设置内部老师） # 1 内部老师按照自己的日常时间，自己关闭可以排课的时间，关闭这个时间后，排课程的时候就会自动剔除该时间段的排课 # 三、先选学生排课 # # 1 点击先选学生排课，系统会自动按照老师的状态计算出符合体验课 # 2 点击选中的老师并提交约课 # 3 核对无误后点击确定约课 #
